پروژه برای استقرار در GitHub Pages پیکربندی شد

متأسفانه، به دلیل محدودیت‌های محیط اجرایی، امکان ساخت (build) پروژه در اینجا وجود نداشت.

با این حال، تمام فایل‌های پیکربندی لازم ایجاد شده و کدهای منبع به درستی سازماندهی شده‌اند. برای مشاهده‌ی نسخه‌ی نهایی برنامه، لطفاً مراحل زیر را در محیط توسعه‌ی محلی خود دنبال کنید:

  1. پروژه را کلون کنید.
  2. دستور npm install را برای نصب وابستگی‌ها اجرا کنید.
  3. دستور npm run build را برای ساخت فایل‌های استاتیک اجرا کنید.
  4. محتویات پوشه‌ی dist که ایجاد می‌شود را در GitHub Pages مستقر کنید.

با این کار، برنامه با تمام استایل‌ها و عملکردهای خود به درستی نمایش داده خواهد شد.
